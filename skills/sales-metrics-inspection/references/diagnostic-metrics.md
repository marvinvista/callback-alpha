# Diagnostic Metrics

## Quantity metrics

- Calls
- Emails
- Meetings
- Opportunities created
- Pipeline value
- Deals won

## Quality metrics

- Connect rate
- Response rate
- Win rate
- Average deal size
- Stage conversion
- Stage age
- Time between touches

## Diagnosis ladder

1. Is there enough activity?
2. If yes, is the quality of that activity strong enough?
3. If yes, is the target or segment wrong?
4. If yes, is deal management weak?
5. If yes, is the issue pricing, packaging, or product fit?
