# Onboarding Plan

## Suggested structure

### Pre-work

- product primer
- market context
- recorded strong calls
- recorded weak calls

### Week 1

- culture and expectations
- market and buyer context
- product walkthrough
- tools and process setup
- scripted repetitions

### Week 2

- mock calls
- shadowing
- supervised live reps
- first KPI check

## Drill ideas

- elevator pitch repetition
- discovery question practice
- objection handling
- demo sections
- handoff and follow-up practice

## Ramp checkpoints

- Can explain the problem clearly
- Can explain the buyer clearly
- Can run the basic workflow
- Can handle common objections
- Can log notes and next steps cleanly
