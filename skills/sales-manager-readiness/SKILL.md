# Sales Manager Readiness

## Overview

Use this skill to decide whether the business is ready to add a dedicated sales manager and what kind of manager would actually fit the current stage.

The job is to avoid premature handoff while still recognizing when seller-led management has become the bottleneck.

## When to Use

- Use when the user asks if it is time to hire a first sales manager.
- Use when a current leader or early seller is still managing reps and is unsure whether to keep doing so.
- Use when the team needs a manager profile, scope, and handoff plan rather than a generic senior sales title.

## When Not to Use

- Do not use when the sales motion has not yet produced repeatable results with multiple sellers.
- Do not use when the need is really to hire another AE, SDR, or CS rep rather than add management.
- Do not use when the user only wants a recruiting spec; route to `sales-hiring-profile` after readiness is established.

## Inputs

- Number of current sellers and roles
- Evidence of repeatable selling
- Current manager or seller involvement
- Metrics and process maturity
- Hiring and onboarding readiness

## Workflow

1. Clarify the current state of the motion:
   - who is selling
   - who is managing
   - what results are repeatable
2. Test readiness using the checklist in `references/readiness-checklist.md`.
3. Look for evidence that the motion can be taught, measured, and monitored without guessing.
4. Distinguish between:
   - need for more individual contributors
   - need for team leads
   - need for a true manager
5. If the answer is “not yet,” state what exit criteria must be met first.
6. If the answer is “yes,” define the right manager profile:
   - tactical and hands-on
   - current-scope fit
   - likely source companies or contexts
7. Recommend a handoff plan and first 90-day scope.

## Output Contract

Return:

- `Readiness verdict`
- `Why`
- `Missing exit criteria`, if not ready
- `Recommended manager profile`, if ready
- `Scope of role`
- `First 90-day priorities`
- `Handoff risks`

## Quality Bar

- Avoid using impressive titles as a substitute for readiness.
- Do not outsource undefined sales motion problems to a new manager.
- Favor manager profiles that match the current scale and selling complexity.
- Make the “not yet” case concrete and constructive.
- Mirror the user’s org terms and role labels; do not introduce unprompted company-stage labels or management labels the user did not use.
