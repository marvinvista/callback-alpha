# Pipeline Review

## Weekly review questions

- Which deals have no clear next action?
- Which deals have gone quiet too long?
- Which deals are low-quality but still consuming attention?
- Which deals deserve concentrated effort this week?

## Stage exit questions

- What evidence justifies this stage?
- What must happen for the deal to move?
- Who owns that next action?
- When will it happen?

## Close-lost categories

- No pain
- Wrong timing
- No budget
- Lost to incumbent or competitor
- Feature or workflow gap
- No decision
- Wrong stakeholder access

## Note-taking prompts

- Why did the deal stall?
- What should future follow-up mention?
- What changed since the last meaningful conversation?
- What would make this worth reopening?
